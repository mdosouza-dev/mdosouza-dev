# Matheus Souza

**`Desenvolvedor FullStack`**

Sou Desenvolvedor Full-Stack em transição de carreira, com sólida base em Tecnologia da Informação, infraestrutura e suporte técnico.

Atualmente estudo e desenvolvo projetos utilizando HTML, CSS, JavaScript, React, Node e Java com foco na criação de sistemas responsivos, funcionais e com boa experiência do usuário.

🎯 Busco oportunidades como Desenvolvedor Full-Stack Júnior, onde eu possa aplicar meus conhecimentos técnicos, evoluir continuamente e contribuir com soluções de qualidade.

📌 Tecnologias: HTML5 | CSS3 | JavaScript | Java | Node | React | Git | GitHub
